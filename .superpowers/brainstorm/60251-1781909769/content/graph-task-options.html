노드 = goroutine. 플레이헤드 시점의 "현재 task 소속"으로 표현합니다. 두 방식 중 선택(또는 둘 다).

C1. task 색 링 (tint)

노드 색(상태)은 그대로, 테두리 링 색 = 현재 task. task 없으면 링 없음. 가볍고 명확.

force가 같은 task 노드를 한곳에 모으고 둘레에 라벨 hull. 강렬하지만 작업량 ↑ (시점마다 소속 변하면 재배치).
